**ASS 04**

This is a simple game made with panda3d and libbulet
*how to compile*

- make (standard mode)
- make DEBUG (to enable debug mode [f1,f2,f3,f4])

*how to use*

- WASD to change position
- Arrow Key to change camera view
- Space Bar to shot
- r to relocate the object (only position no physics)
